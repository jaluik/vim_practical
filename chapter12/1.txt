vim全局设置忽略大小写 
:set ignorecase

vim每次查找时忽略大小写
在查找模式下，输入\c可以忽略大小写  \C强制区分大小写

vim更智能的大小写敏感性设置


body{ color: #3c3c3c; }
a { color: #0000EE; }
strong { color: #000;}

vim中默认 { 需要转义  (需要转义  | 也需要转义
[ 不需要转义
比如上面想要匹配的话，可以使用 /#\([0-9a-zA-Z]\{6}\|[0-9a-zA-Z]\{3}\)

=======================
 
very magic模式

可以使用\v 来激活very magic搜索模式
very magic 模式假定除 _, 大小写字母以及数字以外的所有字符都具有特殊含义
比如 上面的正则可以重写为
/\v#([0-9a-zA-Z]{6}|[0-9a-zA-Z]{3})

其中[0-9a-zA-Z]可以使用 \x来替代
因此可以简写为
/\v#(\x{6}|\x{3})
